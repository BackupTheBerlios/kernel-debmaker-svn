kernel-debmaker


What is it?

A tool for building .deb's of a kernel automatically. Theorically, you just have to edit a configuration file and run a command as root...

Features

What kind of tasks can this stuff do?
  • fetch sources and patch them thanks to the "ketchup" script, and create a place where you can make your kernel configuration (i.e. create a .config file).
  • build .deb packages and an apt repository.

Requirements

Required before running "make":
  • apt-get install kernel-package
  • use "apt-cache search kernel-patch-debian" to show the available debian patches)
  • apt-get install kernel-patch-debian-x.x.x
  • edit your /etc/kernel-pkg.conf or ~root/.kernel-pkg.conf and make sure you have a "image_in_boot := True" line in it. Add it if it's not here.
  • get the "ketchup" script ( http://www.selenic.com/ketchup/ ), which requires python
  • fill the config section of the Makefile
  • fasten your seatbelt and run "make" as root ;)

Make targets

  • all
    Fetches the sources, builds the .deb packages of the kernel, and creates an apt repository for them.
  • fetch
Fetches the kernel sources thanks to the ketchup script.
  • clean
Cleans the installed sources tree (i.e. runs "make clean" in the sources tree).
  • clean-binary
Removes the apt repository and the sources tree.
